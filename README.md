# Fallout-Badge

![picture of board](/Images/board.png)

My badge for fallout, its great

This is my badge for use at Fallout! Its silly and goofy like me. It has a chaotic LED circuit based on (this)[https://www.creative-science.org.uk/chaotic_leds.html] one. I wanted something fun so it plugs into your PC then does something cool. It just blinks randomly based on some 555 timers being connected together.
